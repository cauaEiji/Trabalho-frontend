# Projeto Web

## Instalação

1. Clone ou baixe este repositório:

```bash
git clone https://github.com/cauaEiji/Trabalho-frontend.git
```

2. Mantenha os nomes dos arquivos conforme estão no projeto.

## Estrutura de Pastas

Organize os arquivos da seguinte forma:

```
/projeto
  ├── index.html
  ├── css/
  │     └── style.css
  └── img/
        └── (imagens do projeto)
```

* Crie uma pasta chamada **css** para os arquivos de estilo
* Crie uma pasta chamada **img** para armazenar as imagens

## Como Executar

1. Abra o projeto no VS Code
2. Instale a extensão **Live Server**
3. Clique com o botão direito no `index.html`
4. Selecione **"Open with Live Server"**

## Deploy

O projeto está sendo executado no **GitHub Pages**, utilizando a branch:

```
develop
```
